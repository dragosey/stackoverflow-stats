You can now track your Stack Overflow reputation and badges directly on your desktop.
After you install it you have to open Stack Overflow Stats.ini file in a text editor and edit these variables:

* soURL = you have to insert the URL to your Stack Overflow profile
* PercentageWidth = this value represent how much width of the screen will represent the width of the skin (value represent %, so please insert value between 0-100)

All set. You are ready to go!
